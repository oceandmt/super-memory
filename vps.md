# VPS Deployment Template

This repository does not store private VPS connection details.

Set these environment variables locally before running deployment scripts:

```bash
export VPS_HOST="<VPS_USER>@<VPS_HOST>"
export VPS_PORT="<VPS_PORT>"
export REMOTE_ROOT="<REMOTE_SUPER_MEMORY_PATH>"
```

Do not commit real hostnames, IP addresses, usernames, SSH key paths, tokens, or private OpenClaw paths.
